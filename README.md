# Social Media Usage and Student Well-Being Dashboard

## Project Overview
This project analyzes the relationship between social media usage, sleep habits, mental well-being, and academic performance using Excel data analysis tools and an interactive dashboard.

---

## Dataset Features
The dataset includes:

- Age  
- Gender  
- Daily Social Media Hours  
- Platform Usage  
- Sleep Hours  
- Screen Time Before Sleep  
- Academic Performance  
- Physical Activity  
- Social Interaction Level  
- Stress Level  
- Anxiety Level  
- Addiction Level  

---

# Data Cleaning
Performed preprocessing steps before analysis:

- Removed duplicates  
- Checked and handled missing values  
- Verified numeric and categorical data types  
- Standardized labels (Gender, Platform Usage, Social Interaction)  
- Converted raw data into Excel Table format for dynamic analysis

---

# Advanced Formatting
Applied formatting techniques for readability and presentation:

- Styled dataset headers  
- Cell borders and color themes  
- Number formatting for averages and KPIs  
- Consistent fonts and dashboard theme  
- Dashboard layout alignment and section grouping

---

# Conditional Formatting
Used conditional formatting to highlight patterns:

- High Stress / Anxiety levels  
- Low Sleep Hours  
- High Addiction scores  
- Academic performance bands  
- Data bars and color scales for comparison

- Anxiety varies slightly across gender groups.
